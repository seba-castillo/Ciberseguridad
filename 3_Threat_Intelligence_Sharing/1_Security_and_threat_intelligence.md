# Security Intelligence

La inteligencia de seguridad es el proceso mediante el cual se generan datos en el uso continuo del sistema de información. Estos datos son recogidos, procesados, analizados y difundidos para proporcionarnos información sobre el estado de seguridad de esos sistemas.

¿Cómo estan nuestros sistemas?

Por ejemplo: Logs de mi firewall.

# Cyber Threat Intelligence

La inteligencia de ciberamenazas es el proceso de investigar, recopilar, analizar y difundir información sobre las amenazas emergentes y las fuentes de amenazas para proporcionar datos sobre el panorama de amenazas externas.

¿Cómo está el panorama general?

Esta información puede llegar de dos formas:

- Informes: Análisis de grupo adversario o de un determinado tipo de malware etc.
- Datos: Puede ser una lista de malos indicadores conocidos, como indicadores de compromiso, nombres de dominio, hashes de código de malware, direcciones IP, etc

Es recomendable utilizar ambas, se pueden pagar a modo de subscripción por ejemplo a McAfee, FireEye, Red Canary, entre otras.
